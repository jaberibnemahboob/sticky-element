Example # 1

This simple example will show how 'Sticky Element' script can be use to make a fixed top navigation bar.

let topMenuBar = new StickyElement({
    element: ".stickynav", /* selector name of the html element, that you want to stick */
    stickTo: 0, /* top offset of the position */
    scrollTowards: "none" /* in which scroll event will trigger to stick the element.
                          Possible 3 values are - 'down', 'up' and 'none' */
});
