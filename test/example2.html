Example # 2

This simple example will show how 'Sticky Element' script can be use to make a fixed top navigation bar with an to offset and the sticky effect will only appear when scrolling down.

let topMenuBar = new StickyElement({
    element: ".stickynav",
    stickToX: 0,
    stickToY: 0,
    scrollTowards: "down"
});
  • 'element' is the selector name of the html element
  • 'stickToY' is the top offset of the position
  • 'scrollTowards' which scroll movement will trgger to stick the element. Values can one of these three "none", "down" and "up".
